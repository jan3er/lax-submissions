# lax-submissions

Lax archive submissions, ported from Édouard Bonnet's twin-width
formalization (github.com/EdouardBonnet/leaning, `twin-width/`, MIT).

- `twin-width-treewidth-exponential/` — **Lax1**: twin-width can be
  exponential in treewidth (three concepts: treewidth, twin-width, and the
  separation theorem).
- `twin-width-mixed-minor-equivalence/` — **Lax2**: twin-width and mixed
  minor number are functionally equivalent (two concepts: mixed minor number
  and the equivalence theorem).

Each folder is a lax submission (`lax build` must pass); see the manifest
and `abstract.md` inside.
